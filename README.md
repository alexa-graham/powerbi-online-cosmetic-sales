# powerbi-online-cosmetic-sales
Interactive sales dashboard that tracks revenue trends, top products, and regional performance using Power BI and DAX

# Tools used:
- Power BI
- Power Query
- DAX

# Dataset:
- 3 years of simulated ecommerce data for a cosmetic retailer
- fields include order date, order ID, customer name, customer city of residence, product category, product type, quantity, price, and payment method

# Key metrics (KPIs):
- Total revenue
- Total orders
- Total quantity sold
- Average order value

# Dashboard features:
- Revenue trends by year
- Revenue trends by product category
- Payment method analysis with interactive slicers
- Multi-page layout
